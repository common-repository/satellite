Satellite is a simple WordPress optimization reporting tool. Get actionable metrics you can use to improve your website's performance.

== Description ==

Satellite is an optimization plugin which reviews your site for easy ways to instantly improve your WordPress website's performance.  These recommendations include metrics such as updating plugins, removing inactive themes, emptying spam comments, using a CDN, and known plugin conflicts.

Checks:

* Poor Rated Plugins
* Unsupported Plugins
* Post revisions
* Host
* SEO plugin
* Trash size
* Spam comments
* Search engine visibility
* Outdated plugins
* Outdated WordPress version
* SSL certificate
* Inactive themes
* Performance damaging plugins
* Caching solution
* Backup solution
* Host disallowed plugins
* Inactive plugins
* Number of active plugins
* DNS records
* Email records
* Website size
* Image optimization
* JS, CSS optimization



== Installation ==

This section describes how to install the plugin and get it working.


1. Upload the plugin files to the `/wp-content/plugins/` directory, or install the plugin through the WordPress plugins screen directly.
2. Activate the plugin through the 'Plugins' screen in WordPress
3. Go to Satellite.fm and enter your email address and the website domain.
4. You will get a report on how to better optimize your site.  Free.
